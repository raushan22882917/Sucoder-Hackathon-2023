Coding Instructions

  1. Read the Prompt Carefully:

    Take the time to thoroughly understand the problem statement. Identify the input, output, and any constraints provided.

    Example: Read the prompt below and analyze the requirements: 

    
              Prompt: Write a function to calculate the factorial of a number.

  2. Plan Your Approach:

    Before coding, outline your approach to solving the problem. Break the problem into smaller sub-problems if possible.

    Example: Outline your approach in pseudocode: 

    
              Pseudocode:
              function calculateFactorial(number):
                if number is 0:
                  return 1
                else:
                  return number * calculateFactorial(number - 1)

  3. Write Pseudocode:

    Draft pseudocode to outline the logic of your solution. Ensure that your pseudocode aligns with the problem requirements.

    Example: Use pseudocode to plan your solution before coding. 

    
              Pseudocode:
              function calculateFactorial(number):
                if number is 0:
                  return 1
                else:
                  return number * calculateFactorial(number - 1)

  4. Test Extensively:

    Test your code with a range of inputs, including boundary cases. Check for edge cases, large inputs, and unexpected scenarios.

    Example: Test your function with various inputs: 

    
              Test Cases:
              calculateFactorial(0) // Expected: 1
              calculateFactorial(5) // Expected: 120
              calculateFactorial(3) // Expected: 6

  5. Document Your Code:

    Provide clear comments explaining the logic behind your code. Make your code readable and understandable by others.

    Example: Add comments to explain your function. 

    
              JavaScript Code:
